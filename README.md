## Setup

To set up this project locally, follow these steps:

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```

2. Navigate to the project directory:
    ```sh
    cd Natours
    ```

3. Install dependencies:
    ```sh
    npm install
    ```

4. Run the development server:
    ```sh
    npm start
    ```
